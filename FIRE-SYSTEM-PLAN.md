# FIRE 財務自由規劃系統 — 業務溝通版

> 讓台灣業務人員擁有一套專業的 FIRE 財務規劃工具，與客戶面對面時，3 分鐘算出財務自由路徑，提升專業形象與成交率。

---

## 🎯 願景與目標

### 願景
**「讓每位業務都能成為客戶的財務自由教練」**

台灣有超過 30 萬名保險業務、理財專員、銀行理專。他們每天都在跟客戶談「退休」「儲蓄」「投資」，但缺乏一套**視覺化、互動式、台灣在地化**的 FIRE 計算工具。

這套系統不是給理財專家用的——是給**第一線業務人員**用的。

### 核心目標

| # | 目標 | 衡量指標 |
|---|------|----------|
| 1 | **3 分鐘完成 FIRE 計算** | 從輸入到產出報告 ≤ 3 分鐘 |
| 2 | **業務秒上手** | 零培訓，拿到就會用 |
| 3 | **客戶看得懂** | 非專業人士也能理解視覺化報告 |
| 4 | **台灣在地化** | 勞保/勞退/健保/所得稅全部內建 |
| 5 | **促進成交** | 計算結果自然導向保險/投資解決方案 |

### 目標用戶

| 用戶角色 | 痛點 | 我們提供 |
|----------|------|----------|
| **保險業務員** | 跟客戶談退休只能用嘴巴算 | 視覺化退休缺口分析 |
| **銀行理專** | 每次都要用 Excel 慢慢算 | 即時互動計算器 |
| **投資顧問** | 客戶對數字無感 | 動態圖表 + 情境比較 |
| **房仲業務** | 客戶猶豫買房影響退休 | 買房 vs 租房 FIRE 比較 |
| **直銷/保經代** | 缺乏專業工具建立信任 | 一鍵產出 PDF 報告 |

| 退休缺口分析 | 年金 + 勞退 vs 退休支出的差額 |

**業務溝通流程設計**：
```
Step 1: 破冰
  「我幫你算一下，看看你幾歲可以財務自由？」
  → 引導客戶輸入基本資料

Step 2: 震撼
  「按照目前的情況，你 X 歲才能退休，而且每月會有 Y 萬的缺口」
  → 顯示退休缺口的紅色警示圖表

Step 3: 轉機
  「如果我們做一點調整...」
  → 情境比較：加保 / 定投 / 節省 的效果對比

Step 4: 行動
  「你看，只要每月多存 X 元，就能提早 Y 年退休」
  → CTA：立即規劃 / 預約詳談 / 下載報告
```

│ 配偶收入     │     │ 孝親費       │     │ 負債         │     │ 風險屬性     │
│ 勞保年資     │     │ 教育費       │     │ 儲蓄險       │     │              │
